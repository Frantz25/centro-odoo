# centro-odoo
centro-odoo es la plataforma de gestión integral para El Manzano, diseñada para ordenar y escalar la operación diaria: reservas inteligentes de espacios y actividades, ventas y facturación de planes/membresías, gestión comercial de clientes y convenios, control de inventario y compras, organización de RR.HH. y turnos, y analítica clave de ocupación e ingresos para decidir mejor.

Versión técnica

centro-odoo
Conjunto de módulos Odoo para El Manzano que cubre:

1. Reservas y calendario de espacios/actividades.
2. Ventas y facturación.
3. CRM para clientes y convenios.
4. Inventario y compras.
5. RR.HH.
6. Reportes operativos de ocupación e ingresos.
